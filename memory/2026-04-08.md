## Bug-Fix: 1 Satz Kraft-Übungen
**Problem:** Kraft-Übungen mit 1 Satz wurden als Zeit-Übungen angezeigt
**Ursache:** Fehlerhafte Heuristik in renderWorkoutsList():
```javascript
// Alt (falsch):
const isTimeBased = ... || (duration===0 && weight===0 && sets<=1 && reps>0);
// Wenn 1 Satz × 10 Wdh × 0kg → wurde als Zeit-Übung erkannt

// Neu (korrekt):
const isTimeBased = (w.exercise_type === 'time') || (duration > 0 && weight === 0);
// Nur wenn explizit exercise_type='time' ODER duration_seconds > 0
```
**Commit:** 6f68365
